
Our project is composed of the following

1- An .ipynb file (attached)

2- A GENRE folder with Sample Songs of different genre for training the model
3- JSON file for storing MFCC Values of the songs in Genre folder

4- A RANDOM folder with unsorted songs
5- JSON file for storing MFCC Values of the songs in Random folder

6- A Sorted Playlist folder

7- Test Images
